Fly a drone with your mind!

The Human-Technology Interaction Laboratory at The University of Alabama presents a web-based simulator of a brain-drone race. Anyone can practice using their brain to control a drone in a lap-based environment.

Play Now

Experience Brain-Drone Interaction

Use your mind to control a drone's speed in a lap-based environment. Keep calm and become engaged to go faster. Try to run all three laps in the least amount of time!

Competitive BCI technology

The use of EEG data from users provides an innovative control mechanism in which anyone can participate. Start the Brain-Drone Race experience to compete against friends.

Software

Unity Technologies

The game engine behind this simulator empowers the drone and landscape models to run smoothly in the web.

WebAssembly

Assembly-like language that runs with near-native performance, designed to run alongside JavaScript. Enables for a very fast execution of the code-base.

BCI.js

Powered by the Node.js library that allows for the creation of web-based BCI applications.
